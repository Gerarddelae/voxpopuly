<h1 align="center">VoxPopuly voting software</h1>

VoxPopuly es una plataforma web para gestionar procesos electorales locales organizados por eventos y puntos de votación. Permite a administradores crear eventos, configurar puntos de votación con delegados, registrar votantes autorizados y gestionar candidaturas en planchas. Proporciona una experiencia de voto simple y anónima para votantes y paneles de estadísticas en tiempo real para delegados y administradores.

Stack técnico
 - Frontend: Next.js (app router), React y TypeScript.
 - Estilos: Tailwind CSS (configurado en `tailwind.config.ts`).
 - Backend/BD: Supabase (Postgres) para autenticación, base de datos y Realtime.
 - Integración: `lib/supabase` contiene clientes y utilidades para conectar con Supabase.
 - Otras herramientas: Node.js, Vercel/entorno serverless para despliegue.
